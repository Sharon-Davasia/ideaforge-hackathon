# Knowledge Files Used in IdeaForge

## 1. Project Templates (project_templates.txt)

Contains:
- MVP timeline templates (2-20 weeks based on complexity)
- Team size guidelines by project type
- Phase distribution percentages
- Sprint structure recommendations
- Effort estimation guidelines
- Common roles needed

## 2. Validation Criteria (validation_criteria.txt)

Contains:
- Market opportunity scoring rubric (1-10)
- Technical feasibility scoring rubric (1-10)
- Competitive landscape scoring rubric (1-10)
- GO/NO-GO decision matrix
- Risk categories to evaluate

## 3. Best Practices (best_practices.txt)

Contains:
- User story format guidelines
- Acceptance criteria rules
- MVP definition principles
- Feature prioritization (MoSCoW)
- Architecture principles
- Technology selection criteria
- Common tech stacks by use case
